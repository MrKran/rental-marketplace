# Alash - Образовательная платформа

## О проекте

**Alash** - это инновационная образовательная платформа для аренды товаров и предоставления образовательных услуг, созданная специально для учеников школ Binom (BI group) в Астане.

## Основные возможности

### Для студентов
-  **Аренда товаров** - ноутбуки, учебники, калькуляторы и другие учебные материалы
-  **Образовательные услуги** - курсы, репетиторство, подготовка к ЕНТ
-  **Система рейтингов** - надежные отзывы и оценки
-  **Чат и сообщения** - общение с продавцами и преподавателями
-  **Уведомления** - отслеживание статуса аренды и бронирований
-  **Избранное** - сохранение понравившихся товаров и услуг

### Для преподавателей
-  **Создание курсов** - предложение образовательных услуг
-  **Управление учениками** - отслеживание записей на курсы
-  **Монетизация** - получение дохода от преподавательской деятельности
-  **Статистика** - просмотр рейтингов и отзывов

## Технологический стек

- **React** - библиотека для создания пользовательского интерфейса
- **TypeScript** - типизированный JavaScript
- **Tailwind CSS v4.0** - утилитарный CSS фреймворк
- **Shadcn/ui** - коллекция компонентов UI
- **Motion (Framer Motion)** - библиотека для анимаций
- **Lucide React** - иконки
- **Sonner** - toast уведомления

## Система безопасности

Платформа включает комплексную систему безопасности:
-  **Валидация форм** - защита от некорректных данных
-  **Санитизация HTML** - защита от XSS атак
-  **Rate limiting** - защита от спама
-  **Логирование** - отслеживание подозрительной активности
-  **Проверка прав** - контроль доступа к функциям

## Начало работы

### Регистрация
1. Нажмите "Войти / Регистрация" в шапке сайта
2. Выберите тип пользователя (студент/преподаватель)
3. Заполните форму регистрации
4. Выберите свою школу Binom
5. Создайте аккаунт

### Добавление товара или услуги
1. Войдите в систему
2. Нажмите кнопку "Добавить" в шапке
3. Выберите тип (товар или услуга)
4. Заполните информацию
5. Опубликуйте объявление

### Аренда или бронирование
1. Найдите интересующий товар или услугу
2. Нажмите "Арендовать" или "Забронировать"
3. Выберите период аренды
4. Оформите оплату
5. Свяжитесь с владельцем через чат
